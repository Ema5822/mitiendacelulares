Proyecto listo para Netlify.

Qué cambia en esta versión:
- Toma las fotos desde la misma página de Evophone.
- Intenta obtener el stock real disponible desde la publicación original.
- Si la página no expone el número exacto, muestra Disponible / Sin stock.
- Mantiene carrito, filtros, detalles y WhatsApp.

Cómo subirlo:
1. Subí la carpeta completa a Netlify.
2. Netlify detecta la carpeta netlify/functions.
3. Ejecutá npm install para instalar cheerio.
4. Publicá el sitio.

Nota:
El número exacto en stock depende de que la web de origen lo exponga públicamente.
